## Project Overview
This project is an Excel-based annual sales data analysis dashboard created to
analyze sales performance, customer demographics, and order trends for UrbanMart Store.
The dashboard converts raw data into meaningful business insights using interactive visuals.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters
- Basic Excel Formulas

## Dashboard Insights
- Order Status: Delivered, Cancelled, Returned, Refunded
- Sales by Gender: Male vs Female
- Monthly Order vs Sales Trend
- Orders by Sales Channel (Amazon, Flipkart, Myntra, Ajio, Meesho, etc.)
- Age & Gender-wise Order Analysis
- Top 5 States by Sales Performance

## Business Purpose
This dashboard helps in understanding sales trends, customer behavior,
and channel-wise performance to support data-driven decision making.
